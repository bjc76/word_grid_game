# Word Grid Game
A dynamic word finding game for two players which involves finding words from a grid of letters before the other player. 

## Features
* **Gameplay** Traditional grid-based word guessing mechanics.
* **Scoring system** Scores awarded based on word length.
* **Letter locking system** Letters are temporarily blocked to the other player when used in a word submitted.

## Language
Java with Vaadin framework, Mavin build tool and Docker.
